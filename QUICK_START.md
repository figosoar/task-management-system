# 快速启动指南

## 🚀 立即开始

### 1. 启动服务器
```bash
npm start
```

### 2. 访问应用
打开浏览器访问: http://localhost:3000

### 3. 管理员登录
- 用户名: `admin`
- 密码: `admin123`

### 4. 进入管理后台
登录后点击右上角"管理后台"按钮

## 📋 核心功能速览

### 管理员可以做什么？
1. **创建用户** - 为团队成员创建账户
2. **分配任务** - 给用户分配工作任务
3. **查看统计** - 监控任务完成情况

### 普通用户可以做什么？
1. **创建任务** - 点击列顶部的"+"按钮
2. **拖拽任务** - 在三个列之间拖动任务卡片
3. **编辑/删除** - 使用任务卡片上的按钮
4. **自动折叠** - 超过6个任务时自动折叠

## 🎯 典型使用场景

### 场景1: 团队任务管理
1. 管理员创建团队成员账户
2. 管理员分配任务给不同成员
3. 成员登录查看并完成任务
4. 管理员查看完成情况统计

### 场景2: 个人任务管理
1. 注册普通用户账户
2. 创建自己的任务
3. 拖拽任务到不同状态
4. 查看任务统计

## 🔧 常用操作

### 创建新用户（管理员）
1. 进入管理后台
2. 点击"创建用户"
3. 填写信息并提交

### 分配任务（管理员）
1. 在用户卡片上点击"分配任务"
2. 填写任务详情
3. 提交

### 管理任务（普通用户）
1. 点击"+"创建任务
2. 拖拽改变状态
3. 点击✏️编辑
4. 点击🗑️删除

## 📊 查看统计

### 管理员统计
- 进入管理后台 → 数据统计标签
- 查看每日完成情况
- 查看用户任务分布

### 用户统计
- 查看底部统计面板
- 显示未完成任务数
- 显示最长停留时间

## 💡 提示

- 任务超过6个会自动折叠，点击"显示更多"展开
- 拖拽任务到"已完成"会自动记录完成时间
- 管理员可以创建其他管理员账户
- 默认管理员账户不能被删除

## ⚠️ 注意事项

1. 首次运行会自动创建数据库
2. 数据保存在 `avengers_todo.db` 文件中
3. 删除数据库文件会重置所有数据
4. 端口3000需要保持空闲

## 🆘 遇到问题？

### 端口被占用
```bash
# 修改 server.js 中的 PORT 变量
const PORT = 3001; // 改为其他端口
```

### 忘记密码
删除 `avengers_todo.db` 文件，重启服务器会重新创建默认管理员账户。

### 页面无法访问
确保服务器正在运行，检查控制台是否有错误信息。
