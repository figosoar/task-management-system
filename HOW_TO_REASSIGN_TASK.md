# 如何重新分配任务

## 快速指南

### 步骤 1: 查看任务详情
1. 登录管理员账户
2. 进入管理后台
3. 切换到"任务管理"标签
4. 点击要重新分配的任务标题

### 步骤 2: 开始重新分配
1. 在任务详情页面
2. 点击左下角的"重新分配"按钮
3. 重新分配对话框会打开

### 步骤 3: 选择新用户
1. 查看当前任务信息
2. 查看当前分配给谁
3. 从下拉列表选择新用户
4. 点击"确认分配"

### 步骤 4: 验证结果
1. 看到"任务重新分配成功"提示
2. 任务列表自动刷新
3. 查看任务的新分配用户

## 示例场景

### 场景：工作量调整

**情况**: 张三任务太多，李四任务较少

**操作步骤**:
```
1. 在任务列表中找到张三的任务
2. 点击任务标题查看详情
3. 确认是张三的任务
4. 点击"重新分配"
5. 选择"李四"
6. 点击"确认分配"
7. 完成！
```

**结果**: 任务从张三转移到李四

## 常见操作

### 操作 1: 快速重新分配
```
任务列表 → 点击标题 → 重新分配 → 选择用户 → 确认
```
**用时**: 约 10 秒

### 操作 2: 批量调整（多个任务）
```
对每个任务重复：
  点击标题 → 重新分配 → 选择用户 → 确认
```

### 操作 3: 撤销分配（分配回原用户）
```
点击标题 → 重新分配 → 选择原用户 → 确认
```

## 提示和技巧

### 💡 提示 1: 查看当前分配
在重新分配对话框中，当前用户会默认选中，方便确认。

### 💡 提示 2: 快速操作
熟练后，可以在几秒内完成重新分配。

### 💡 提示 3: 验证结果
重新分配后，任务列表会自动刷新，立即看到结果。

### 💡 提示 4: 取消操作
如果不想分配，点击"取消"或模态框外部即可。

## 注意事项

### ⚠️ 注意 1: 只能分配给普通用户
下拉列表只显示普通用户，不包括管理员。

### ⚠️ 注意 2: 原用户将失去访问权
任务重新分配后，原用户无法再看到该任务。

### ⚠️ 注意 3: 任务状态保持不变
重新分配不会改变任务的状态、优先级等属性。

### ⚠️ 注意 4: 需要管理员权限
只有管理员可以重新分配任务。

## 故障排除

### 问题 1: 看不到"重新分配"按钮
**原因**: 不是管理员账户  
**解决**: 使用管理员账户登录

### 问题 2: 下拉列表为空
**原因**: 没有普通用户  
**解决**: 先创建普通用户

### 问题 3: 提交后没有反应
**原因**: 网络问题或未选择用户  
**解决**: 检查网络，确保选择了用户

### 问题 4: 提示"无权限"
**原因**: Session 过期  
**解决**: 重新登录

## 最佳实践

### 1. 合理分配
- 考虑用户的工作量
- 考虑用户的技能
- 避免频繁调整

### 2. 及时沟通
- 重新分配前与相关人员沟通
- 说明分配原因
- 确认新用户能够接手

### 3. 记录原因
- 在任务描述中添加备注
- 说明为什么重新分配
- 便于后续追溯

### 4. 定期检查
- 定期查看任务分配情况
- 及时调整不合理的分配
- 保持工作量平衡

重新分配任务的完整流程：
```
1. 任务管理 → 点击标题
2. 任务详情 → 重新分配
3. 选择用户 → 确认分配

```

简单、快速、高效！
